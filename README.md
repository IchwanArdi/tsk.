# tsk.

A brutalist-minimal task manager built with **React**, **TypeScript**, and **Tailwind CSS**. No noise, just focus.

![Preview Image Placeholder](https://via.placeholder.com/800x450.png?text=tsk.+Project+Preview)

## ⚡ Features

- **Brutalist Aesthetic**: Dark monochrome theme with high-contrast electric lime accents.
- **Micro-animations**: Staggered list entry, animated custom checkboxes, and smooth delete transitions.
- **Type-Safe**: Built with strict TypeScript fundamentals to ensure reliability and catch errors early.
- **Task Management**: Filter tasks by All, Active, or Completed states.
- **Real-time Clock**: Integrated dashboard-style clock for better time awareness.
- **Persistent Storage**: Automatically saves your tasks to `localStorage`.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Font**: Space Grotesk & JetBrains Mono

## 🚀 Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/IchwanArdi/tsk..git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
